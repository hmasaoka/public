# Claude Code セキュリティ設定ナビゲーター アプリ 要件定義

## 概要

非エンジニア向けに、Claude Code のセキュリティ設定を対話形式（質問形式）で案内し、安全な設定を簡単に行えるようにするアプリ。

ユーザーは専門知識がなくても、いくつかの質問に回答するだけで、自分に適した Claude Code のセキュリティ設定内容を理解し、設定手順を確認できる。

---

# 目的

Claude Code のセキュリティ設定について、

- 専門用語を使わずに説明する
- 質問に答えるだけで推奨設定を導き出す
- セキュリティリスクを可視化する
- 設定手順をわかりやすく提示する

ことで、非エンジニアでも安全に Claude Code を利用できるようにする。

---

# 解決したい課題

## 現状の課題

Claude Code のセキュリティ設定には以下のような問題がある。

- 専門用語が多い
- 設定項目の意味が分かりにくい
- 何を選べば安全なのか判断できない
- 誤設定による情報漏洩リスクがある

## 解決後

ユーザーは質問に答えるだけで、

- 現在のリスクレベル
- 推奨設定
- 理由
- 設定方法

を理解できる。

# アプリの基本フロー

## Step1

利用目的を質問

例

「Claude Code を何に使いますか？」

選択肢

- 個人学習
- 趣味開発
- 業務利用
- 顧客案件
- 社内システム開発

---

## Step2

扱うデータを質問

例

「機密情報を扱いますか？」

選択肢

- 扱わない
- 少し扱う
- 頻繁に扱う
- 分からない

---

## Step3

実行権限を質問

例

「AIによるコマンド実行をどこまで許可しますか？」

選択肢

- 毎回確認したい
- 基本確認したい
- よく使うものだけ自動許可
- すべて許可

---

例

選択肢

- 作業フォルダのみ
- プロジェクト全体
- PC全体
- 分からない

---

## Step5

インターネット利用を質問

例

「Claude Code の外部通信を許可しますか？」

選択肢

- 不要
- 必要時のみ
- 常時許可
- 分からない

---

## Step6

例

- 分からない

# 判定ロジック

## リスクレベル

### 低リスク

条件

- 個人利用
- 機密情報なし
- 実行前確認あり
- 作業フォルダ限定

表示

🟢 安全性が高い設定です

---

### 中リスク

条件

- 業務利用
- 一部機密情報あり
- 一部自動実行あり

表示

🟡 注意が必要です

---

### 高リスク

条件

- 顧客情報あり
- 全自動実行
- PC全体アクセス
- Push自動実行

表示

🔴 設定の見直しを推奨します
