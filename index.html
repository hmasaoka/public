現状の課題

こんなことで困っていませんか？

Claude Code のセキュリティ設定は、専門知識がないと何をどう設定すればいいか判断が難しい。

📖

専門用語が多い

「パーミッション」「CLI」「権限スコープ」など、意味がわからない言葉が並ぶ

❓

何を選べば安全？

設定項目がたくさんあって、どれが自分に必要なのか判断できない

😰

誤設定が怖い

間違えたら情報漏洩につながるかもと思うと、設定が怖くて手が止まる

🔍

リスクが見えない

今の設定が安全かどうか、どんなリスクがあるかが全くわからない

↓

セキュリティ設定ナビゲーターが解決します

専門知識ゼロでも OK。質問に答えるだけで、現在のリスクレベルと推奨設定・設定手順をわかりやすく提示します。不安を感じることなく、安全に Claude Code を使い始められます。

特徴

専門知識ゼロで使える4つのポイント

難しい言葉は一切使いません。小学生でもわかる言葉で、あなたに最適な設定を案内します。

💬

選ぶだけで診断完了

文字を入力する必要はありません。質問に対して選択肢を選ぶだけで、あなたに合った設定が導き出されます。

🛡️

リスクを3段階で可視化

現在の設定が「安全」「要注意」「要見直し」のどのレベルか、一目でわかるように表示します。

📋

推奨理由も丁寧に説明

「なぜこの設定が安全なのか」を必ず説明します。理由がわかるので、自分で判断できるようになります。

📄

設定手順をPDF出力

診断結果と設定手順をPDFで保存できます。後から見返したり、同僚と共有したりするときに便利です。

使い方

6つの質問に答えるだけ

順番に選択肢を選んでいくだけで、あなたに最適なセキュリティ設定が完成します。

1

利用目的を教えてください

「Claude Code を何に使いますか？」

個人学習 趣味開発 業務利用 顧客案件 社内システム開発
2

扱うデータについて

「機密情報を扱いますか？」

扱わない 少し扱う 頻繁に扱う 分からない
3

コマンドの実行許可

「AIによるコマンド実行をどこまで許可しますか？」

毎回確認したい 基本確認したい よく使うものだけ自動許可 すべて許可
4

ファイルアクセスの範囲

「Claude Code がアクセスできる範囲は？」

作業フォルダのみ プロジェクト全体 PC全体 分からない
5

インターネット利用

「Claude Code の外部通信を許可しますか？」

不要 必要時のみ 常時許可 分からない
6

Git 操作の許可範囲

「AIによるGit操作を許可しますか？」

読み取りのみ コミットまで Pushまで 分からない
